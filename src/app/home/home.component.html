Welcome

  • This is the user interface (UI) for the laptop procurement process displayed on the right.
  • The UI has 3 views, choose one of the views in the radio list to learn more about each view.
  • Choose Process Overview to return to the process diagram.

Process Overview

Employee View

  • The Employee view allows you to request a laptop.
  • This is the starting point of the procurment process.
  • To access this view click on Employee in the navigation menu.

Procurement Manager View

  • The Procurement Manager view allows you to request an offer from a supplier.
  • You can also manually approve an Offer if it was rejected by the system.
  • To access this view click on Procurement Manager in the navigation menu.

Supplier View

  • The Supplier view allows you to prepare an offer for the request from the Procurement Manager.
  • To access this view click on Supplier in the navigation menu.
